1. Go to the 'backend' folder.
2. Run the command 'npm install' in CLI.
3. Then run the command 'nodemon app' in CLI.
4. Go to 'frontend' folder.
5. Run the command 'npm install' in CLI.
6. Then run the command 'npm start' in CLI.

Student Login:
    Method 1: Create a student account with 'Register' page to login to the system as a student.
    Method 2: Use the following login credentials of an already created student account to login
              to the system as a student.
                  Email    : student@gmail.com
                  Password : student

Teacher Login:
    Use the following login credentials of an already created teacher account to login to the
    system as a teacher.
        Email    : teacher@gmail.com
        Password : teacher
